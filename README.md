# OIRTE AI 🤟

**Aplicación web progresiva de asistencia comunicacional para personas sordas y adultos mayores.**

Desarrollado con Next.js, Gemini AI (gemini-2.5-flash), Web Speech API y TTS.

## Funcionalidades

- 🎤 **Escucha en tiempo real** — Transcribe voces del ambiente en texto grande
- 💬 **Chat conversacional** — Historial tipo mensajería con auto-stop por silencio
- 🤖 **Respuestas IA** — Gemini genera 3 respuestas sugeridas según contexto
- ⌨️ **Respuesta manual** — Escribir y enviar respuestas propias
- 📸 **Análisis de fotos** — Sube una imagen y Gemini la describe
- 📄 **Resumen de documentos** — Sube foto de documento para resumen
- 🎵 **Transcripción de audio** — Sube un audio corto para transcribirlo
- 🔊 **Text-to-Speech** — Lee en voz alta todas las respuestas
- 📱 **Interfaz mobile-first** — Navegación por pestañas tipo app nativa
